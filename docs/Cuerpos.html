    Elementos de los cuerpos finitos

    fp es la lista de los elementos del cuerpo Fp. Por ejemplo,

    >>> let xs = f3
>>> xs
[0,1,2]
>>> :type xs
xs :: [F3]
>>> let xs = f5
>>> xs
[0,1,2,3,4]
>>> :type xs
xs :: [F5]

    Operaciones en cuerpos finitos

    >>> 2*3 :: F5
1
>>> 2*3 :: F4
0
>>> 2+3 :: F5
0
>>> 2*3 :: F5
1
>>> 3^2 :: F5
4
